Body Measurement App

Inspiration

Inspired by the challenges of online shopping and finding the right fit, we developed an app to calculate accurate body measurements using just a smartphone.

What It Does

The app processes images taken by your smartphone camera to generate precise body measurements. It eliminates the need for manual measurements and expensive equipment, revolutionizing online shopping.

How We Built It

We utilized Python for backend image processing, JavaScript for the front-end interface, and machine learning algorithms for accurate measurement calculations. The development involved data collection, model training, and implementing a responsive UI.

Accomplishments

  • Developed accurate image processing algorithms for body measurements.
  • Created a user-friendly, responsive interface.
  • Won 3rd place in a competitive hackathon.

What We Learned

We gained valuable insights into AI implementation, image processing, and teamwork. This project enhanced our understanding of real-world problem-solving with technology.

CamGenX

Inspiration

CamGenX was inspired by the growing demand for custom content creation in the digital space. The goal was to provide an accessible, AI-powered tool for generating unique, context-aware images.

What It Does

CamGenX is an AI-powered image generator that creates personalized, context-specific images based on user input. It's designed for digital creators looking to enhance their content effortlessly.

How We Built It

We integrated pre-trained AI models with Python for backend processing and JavaScript for the user interface. The development process involved implementing a seamless workflow from user input to image output.

Accomplishments

  • Developed a fully functional AI-powered image generator within a short timeframe.
  • Designed an intuitive and engaging user interface.
  • Won 10th place in a competitive hackathon.

What We Learned

This project deepened our understanding of AI-powered tools, front-end and back-end integration, and collaborative development in a hackathon setting.
